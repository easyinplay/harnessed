### D-03: token budget approx 算法 = BUFFER /4 HEURISTIC (sister Phase 3.1 D-01)

**Selected**: BUFFER /4 — `Buffer.byteLength(str, 'utf8') / 4` heuristic

**Locked rationale**: sister Phase 3.1 D-01 enforceBudget 已用 + verified acceptable variance (1 char ≈ 0.25 token English / ~0.5 token/char Chinese); zero-dep (Karpathy YAGNI 1 sister precedent); deterministic + cross-OS. **Implementation hint**:

- Karpathy hard limit: check-token-budget.ts ≤40L

**decision_source**: 用户首轮决策 BUFFER /4 HEURISTIC (Recommended option)
