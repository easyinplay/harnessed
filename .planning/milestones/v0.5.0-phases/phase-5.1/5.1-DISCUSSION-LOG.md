## Q1 (HIGH, deliberate): R10.1 filter expression syntax

**Decision**: D-01 jq pipe 内嵌 invocation

### Alternatives considered

| Option | Pros | Cons | Selected |
|--------|------|------|----------|
| jq pipe 内嵌 invocation | jq Unix 哲学 + 表达力极强 + 工程师 familiar + doctor 3rd check 已 verify zero-new-dep | spawn subprocess 性能 ~5-10ms overhead per filter (negligible for CLI) | ✓ |

**User's choice**: jq pipe 内嵌 invocation (Recommended)
**Notes**: 6 视角 briefing 对齐 — Eng familiar + Designer Unix 哲学 + Paranoid Staff Eng zero-new-dep + CSO no untrusted parser layer
