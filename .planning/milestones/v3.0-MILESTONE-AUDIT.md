## v3.0 lessons learned

1. **Architectural smell catch post-GA value**: v2.0.0 GA SHIPPED 后 24 小时内 user catch `/plan-feature` 5-phase conflate fundamental flaw — post-GA architectural review (NOT pre-GA Q-AUDIT) catches what initial discuss-phase batch 1 misses。Mitigation: post-major-version GA window 24-48h dedicate review session institutionalize 推 v3.x 评估 (sister Phase 2.6 retrospective `/retro` workflow ship 2026-05-21 入 standalone)。

2. **Master orchestrator Hybrid Option C 设计**: 初 Phase 3.2 Wave A research 列 3 option (Option A 串行 serial-only + Option B 并行 parallel-only + Option C Hybrid 5-phase logic gate eval + serial/parallel split + spawnSubWorkflow Path A SDK default + Path B sub-shell fallback), Phase 3.5 W0.1 ship Option C 验证最 robust (sister masterOrchestrator.ts 197L K8 ctx single snapshot + K14 warn-not-halt arbitration)。**Lesson**: 多 option research-driven 选 Hybrid 而非 single option, Karpathy ≤200L 紧贴边界 (197L exact) 但可控。

3. **R8.1 dogfood-first methodology value 实证 + extend**: Phase 3.5 W2.1 4-cycle end-to-end (40 fixture) caught 1 production bug (masterOrchestrator spawn order divergence — schema-shape pass but runtime fail) sister Phase 2.5 uppercase OR/AND benchmark verbatim match。**Lesson**: schema-shape regex test + runtime expr-eval dogfood 两层独立 gate, schema pass NOT 等于 runtime pass; 4-cycle end-to-end real fs.readFile + parseYaml + evalGate dogfood 必跑。

4. **Pattern A 升级到 4-teammate scope expand**: Phase 3.4 verify-teammate W0.13 拆 5 sub 因 Pattern C 4-specialist 升级 ADR 0028 verify 拆 7 sub (sub count 14 → 16); Pattern A 3-teammate baseline scope expand 到 4-teammate 自然成本 ~1.33× (sister agent-teams.md token 估算公式 `team_cost ≈ N_teammates × N_rounds × avg_tokens_per_round + N_teammates × initial_brief_tokens`)。**Lesson**: Pattern A 不一定固定 3-teammate, scope expand per phase complexity adapt — 4-teammate 仍 < 2× subagent fan-out cost, 经济。

5. **5 subagent fan-out vs Pattern A team 选择 (Phase 3.6 close)**: file-independent (3 ADR + README + WORKFLOW.md 5 separate file by 5 separate subagent, no SendMessage cross-deps) + close phase 不需 architectural arbitration — subagent fan-out token cost ~150K vs team ~200K (Pattern A initial brief 3× teammate + send-message rounds overhead)。**Lesson**: Pattern A 仅 justified for API contract cross-deps (Phase 3.3/3.4/3.5 execute scope); close phase 不需 team 升级 (sister CLAUDE.md "升级 team 成本翻倍, 按需启用" 铁律守住)。
