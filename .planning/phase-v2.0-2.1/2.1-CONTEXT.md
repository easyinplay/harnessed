## Implementation Decisions

### Milestone class (M-01)

### D-01 Q1 (RESET) — Pure bundled distribution
- **D-01:** **Pure bundled** — workflows / capabilities / judgment 全部在 npm package readonly `<packageRoot>/workflows/` 下, end-user share-only (NO `~/.harnessed/` user-dir override). 项目最终目的 = ship maintainer 三层栈给其他 user, NOT parse 其他 user 的 CLAUDE.md。user 装齐 harnessed 后立即享用 maintainer 流程, 无需写 CLAUDE.md prose。trade-off accepted: maintainer occasional npm patch release (~2-3 day cycle per upstream / capability adjustment), 不 frequent 改 workflow。
- **Reframe context**: 原 Q1 误识为 "整合 user CLAUDE.md", user 2026-05-20 显式 CORRECT — "我们不是要整合用户的claude.md文件,而是我想把我的claude.md文件里面的三层栈开发流程机器化之后,让其他人装了harnessed之后就可以用上我这套开发流程。" Q1-Q4 RESET 重选。

### D-02 Q2 (RESET) — capabilities.yaml flat yaml map
- **D-02:** capabilities.yaml schema = flat yaml map `capability_name → {impl, cmd, since}`。NOT 嵌套 yaml / NOT JS plugin / NOT dynamic decoration。Flat yaml = 最小学习成本 + TypeBox schema validate 直接 + Pure bundled readonly 适配。

### D-03 Q3 (RESET) — expr-eval npm dep
- **D-03:** `expr-eval` npm package (~5KB MIT, 4M weekly downloads) — workflow.yaml gate 字段 expression 解析 + 求值 一站式。NOT 自写 mini-parser (YAGNI + 安全审计成本) / NOT jsep (jsep 是 parser-only, 需自己 eval AST, 增加 ~100L 不必要)。

### D-04 Q4 (RESET) — single judgment.yaml shared SoT
- **D-04:** `<packageRoot>/workflows/judgment.yaml` = maintainer 三层栈判据机器化 single SoT。schema:
  ```yaml
  triggers:
    strategic-gate:        # gstack /office-hours 触发条件
      fires_when: "phase.type in ['new_project', 'new_milestone', 'new_feature']"
      skips_when: "phase.type in ['bug_fix', 'tech_debt', 'continuing_phase']"
    phase-gate:           # GSD /gsd-discuss-phase 触发条件
      fires_when: "phase.open_decisions >= 2 OR phase.scope_days > 1"
      skips_when: "phase.single_task == true"

      fires_when: "subtask.approaches >= 2 OR subtask.core_algorithm == true"
      skips_when: "subtask.type == 'crud' OR subtask.lines < 20"
  ```
  workflow.yaml gate 字段 reference triggers via `gate: triggers.strategic-gate.fires`。NOT parse user CLAUDE.md (REFRAME) — judgment.yaml 是 maintainer ship 给 user 的 bundled SoT。
