## 4 D-decisions LOCKED (no sneak-in, user 按 recommended 授权 1-pass)

### D-01 AuditFormat: JSONL append-only + 11 字段 routing-decision schema (GA-1)
- **Format**: JSONL (1 line per decision) — machine-parseable (jq filter) + human-readable (grep) + append-safe (no JSON-array rewrite)
- **Schema 11 字段** (per decision):
  - `ts` (ISO 8601 timestamp)
  - `phase` (TaskContext.phaseId, fallback "unknown")
  - `task_excerpt` (truncated 200 char of task.task)
  - `task_sha1` (full task sha1 for dedup + cross-ref)
  - `matched_rule_id` (Rule.id from decision_rules.yaml OR null)
  - `primary_expert` + `secondary_expert` (from ArbitrateResult)
  - `category` (meta/engineering/design/content/testing/search)
  - `route_layer` ("L1-keyword" / "L2-semantic-stub" / "L3-fallback")
  - `outcome` ("complete" / "max-iter" / "verbatim-fail" / "spawn-err" / "install-err" / "arbitrate-err")
  - `session_id` (if captured from SDK system:init; nullable)
  - `iter_count` (ralph-loop iterations to COMPLETE; nullable on fail)
- **Sneak-block**:
  - NO JSON array (rewrite-overhead append-unsafe under concurrent write)
  - NO multi-format dual-write (维护 2x; JSONL 双兼)
  - NO full agentDef 14 字段 dump (敏感 path 用户名 leak + 体积爆炸)
  - NO subagent text/result body (敏感 code leak + 体积)
  - NO MCP secrets / env vars
- **Storage**: `.harnessed/audit.log` (single file, append-only); `.harnessed/` 已存在 + .gitignore
- **Rationale**: R8.1 验收 "路由决策 100% 可追溯" 是 routing layer 追溯, NOT subagent output 追溯; downstream consumer = future `harnessed audit log --filter` 子命令 (推 v0.5+) + grep + jq one-liner
