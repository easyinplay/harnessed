**Notes:** README 用户面 enumeration 保留 visible（用户读 README 看 phase shipped 进度）— 非 SSOT 收敛激进路径 a（删 README 冗余收敛 STATE.md）。 沿袭 Phase 2.2 T0.4 freshness gate yaml pattern，补 counter 维度。 三计数一致校验机械可靠（grep -cE 三段后比对），CI fail 第一时间发现 README 漂移（H3 sister review root-cause-class 第 3 次复发 — Phase 2.2 / Phase 2.3 W3-prereq / Phase 2.3 ship 后；人工 review 不可靠已证）。 yaml CI step ~15L：
```yaml
- name: README counter integrity gate
  run: |
    SHIPPED=$(grep -cE "Phase 2\.[0-9]+ shipped" README.md)
    BARS=$(grep -cE "Acceptance bar [A-Z][0-9]" README.md)
    L44=$(sed -n '44p' README.md | grep -oE '[0-9]+/4' | head -1 | cut -d/ -f1)
    if [ "$SHIPPED" != "$BARS" ] || [ "$SHIPPED" != "$L44" ]; then
      echo "README counter drift: shipped=$SHIPPED bars=$BARS L44=$L44"; exit 1
    fi
```
exact regex 由 plan-phase 实占（grep pattern 校准 — 当前 README 行格式 sample 后 lock，避免误判）。 Wave 0 ship。 alternative-considered: A 路径激进收敛失去 README 用户面 enumeration（用户读 README 习惯），C 路径已证人工 review 不可靠（H3 第 3 次复发）。
